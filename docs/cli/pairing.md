# `mazelclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
mazelclaw pairing list whatsapp
mazelclaw pairing approve whatsapp <code> --notify
```
