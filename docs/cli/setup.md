# `mazelclaw setup`

Initialize `~/.mazelclaw/mazelclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
mazelclaw setup
mazelclaw setup --workspace ~/.mazelclaw/workspace
```

To run the wizard via setup:

```bash
mazelclaw setup --wizard
```
